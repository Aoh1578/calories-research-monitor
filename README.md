# Calories Research Monitor

Static evidence ledger and cron-ingested research monitor for the Calories book project.

- Canonical data: `data/data.json`
- Static site: `docs/`
- Ingestion: `scripts/ingest_entries.py incoming/YYYY-MM-DD-candidates.json`
- Deployment: `scripts/deploy_website.py`

Entries are provisional. Social items are discovery pointers, not evidence. HIGH entries still need human review before becoming book-ready citations.
